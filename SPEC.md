# 天地鲟鳇 网站项目规划

## 项目目标

一个可上线的企业展示官网，带后台内容管理，方便更新内容。

## 技术方案

**推荐方案：Hugo + TinaCMS + GitHub Pages**

| 组件 | 选择 | 理由 |
|------|------|------|
| 前端框架 | Hugo | 快速、静态、性能好 |
| 内容管理 | TinaCMS | 免费、集成简单、可视化编辑 |
| 托管平台 | GitHub Pages | 完全免费、可自定义域名 |

## 功能清单

- [ ] 首页 - 公司介绍、Logo展示
- [ ] 关于我们 - 企业故事
- [ ] 产品展示 - 产品列表与详情
- [ ] 联系我们 - 联系方式表单
- [ ] 后台管理 - TinaCMS 可视化编辑内容

## 学习路径

1. **第一步**：本地构建网站
2. **第二步**：添加后台管理
3. **第三步**：部署到 GitHub
4. **第四步**：绑定自定义域名（可选）

## 预计成本

- 域名：约60元/年（可选，暂时用免费地址）
- 托管：0元（GitHub Pages）
